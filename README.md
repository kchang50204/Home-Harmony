# HomeHarmony

An iOS app for roommates to manage chores, groceries, bills, and reminders.

## App Description
HomeHarmony helps people living together stay organized by keeping track of household responsibilities in one place. Users can add chores, manage grocery items, track bills, convert bill amounts using a live currency API, and create shared house reminders.

## Features
- Dashboard with live summary of chores, grocery items, and bills
- Chores list with completion toggles and hide/show completed filter
- Grocery list with category tags and purchased tracking
- Bills screen with live currency API conversion and per-person split
- House reminders with date, time, and optional notes
- Persistent storage using UserDefaults
- Clean SwiftUI interface with tab-based navigation

## Rubric Coverage
- **3 Screens:** Dashboard, Chores, Grocery List, Bills, House Reminders
- **3 Colors:** Green, Orange, Purple
- **3 Data Types:** String, Bool, Double, Date
- **3 GUI Objects:** Toggle, DatePicker, List
- **Web API:** Live currency exchange via Open Exchange Rates API
- **Persistent Storage:** UserDefaults saves all data across sessions
- **User Experience:** Tab navigation, swipe to delete, completion indicators
- **Usefulness:** Helps roommates organize shared household responsibilities

## Tech Stack
- Language: Swift
- Framework: SwiftUI
- Storage: UserDefaults
- API: Open Exchange Rates

## How to Run
1. Clone the repository
2. Open `HomeHarmony.xcodeproj` in Xcode
3. Select an iPhone simulator
4. Press Run ▶️
5. Use the tab bar at the bottom to navigate
